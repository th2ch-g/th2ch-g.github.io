## 3. リンクポリシー

本サイトへのリンクは原則自由です。事前許諾は不要です。可能な限り `<a>` タグでの直接リンクを推奨します (画像埋め込みでの転載ではなく)。

ただし、本サイトの内容を誹謗中傷する目的、もしくは公序良俗に反する目的でのリンクはお断りします。

外部サイトへのリンクは情報提供を目的としており、リンク先の内容を保証するものではありません。リンク先で生じた問題について運営者は責任を負いません。

## 4. コメント機能について

ブログ記事のコメント欄は [Giscus](https://giscus.app) によって提供されており、コメントは GitHub Discussions に保存されます。コメント投稿には GitHub アカウントが必要です。

以下のコメントは予告なく削除する場合があります。

- 誹謗中傷、差別的表現、ハラスメントを含むもの
- スパム、無関係な広告、外部サイトへの誘導
- 法令違反、または法令違反を助長する内容
- 本人の同意なく個人情報を含むもの

コメントへの返信義務は負いません。

## 5. 推奨環境

本サイトは以下の環境での閲覧を推奨します。

- モダンブラウザの最新版 (Chrome / Firefox / Safari / Edge)
- JavaScript 有効 (検索機能、Mermaid 図、Giscus コメント等で使用)

JavaScript を無効にしても主要コンテンツの閲覧は可能ですが、一部機能が利用できません。
